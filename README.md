# File Management System

## Project Description
This File Management System is a web-based application developed using PHP, MySQL, HTML, CSS, JavaScript, and XAMPP that allows users to manage files and user records efficiently through an interactive dashboard with database integration and responsive design.

---

## Features
- User Management
- File Upload and Management
- Dashboard Interface
- Database Integration using MySQL
- Responsive UI

---

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

---

## How to Run the Project

1. Install XAMPP  
2. Copy the project folder into:
   `C:/xampp/htdocs/`

3. Start:
   - Apache
   - MySQL

4. Import the database:
   - Open phpMyAdmin
   - Create a database
   - Import `file_manage.sql`

5. Open browser and run:
   `http://localhost/file_management`
